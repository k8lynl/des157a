I chose to refine the MadLibs project, because it is my personal favorite studio project and everyone I have sent it to has enjoyed it as well.

Project Description

I have decided to focus on functionality when improving my MadLibs project. Specifically, I want to make this project more engaging and interactive with additional features. For example, besides filling in words in a form, I want users to be able to select the type of font and colors that will be used in their final results. I am planning on also refining the UX of the project a bit as well. For this, I plan on breaking down the form so that it feels more succinct. Currently, the MadLibs has many input fields, which may deter users from finishing the form because it is too long.
